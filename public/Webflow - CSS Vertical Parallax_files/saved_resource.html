Parallax with CSS only

Follow @vinchubang
WebfLow designer page

This 8 layers vertical parallax is made in Webflow with some line of custom CSS code. No Javascript. 

All the images are SVG. The method is long and has some downsides, requires a few fine tuning. And you cant really control the length of the final page. I dont recommend this method for a full production site. Its more an experiment. You can duplicate this site and analyse the settings. Below is the custom CSS code added to the sites dashboard.
